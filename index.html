O que é LexIDE

É uma IDE(ambiente de desenvolvimento) para linguagem Lex, desenvolvida em sala de aula na disciplina de compiladores do curso de Ciência da Computação na Universidade do Vale do Itajaí (UNIVALI) em 2016, pelos acadêmicos Ailton Cardoso Júnior(Ailtonjr) e João Felipe Gonçalves (JFRode). A IDE transcreve da linguagem Lex para o assembly do processador BIP IV, também desenvolvido na UNIVALI. Para saber mais do Processador BIP clique aqui.

A finalidade desta linguagem e IDE é apenas aprendizado, você pode acessar o projeto no GitHub e visualizar a documentação completa do projeto.
