# LexIDE
Projeto da disciplina de compiladores.

Tem como objetivo a criação de uma linguagem de programação simples, apenas para aprendizagem. Inclui uma IDE.

--

###Data types
| Data | Example |
| ---- | ------- |
| int | 16 |
| string | "Hello world!" |
| char | 'c' |
| float | 10.23 |
| vectors | int vet[] |

###Control structures
| Conditional clauses | Example |
| ------------------- | ------- |
| if | if (...) {...} |
| if else | if... else {...} |
